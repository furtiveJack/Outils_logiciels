# ASCII ART

The purpose of this program is to transform a string into its Ascii representation, like this :

```
Hello World
```

-> 

```
 _    _        _  _            __          __              _      _ 
| |  | |      | || |           \ \        / /             | |    | |
| |__| |  ___ | || |  ___       \ \  /\  / /   ___   _ __ | |  __| |
|  __  | / _ \| || | / _ \       \ \/  \/ /   / _ \ | '__|| | / _` |
| |  | ||  __/| || || (_) |       \  /\  /   | (_) || |   | || (_| |
|_|  |_| \___||_||_| \___/         \/  \/     \___/ |_|   |_| \__,_|

```

## Launch the program

Just run : 

```
python3 ascii_art.py "A string here"
```
