# Les dictionnaires

## Généralités

* Ensemble de clés/valeurs

## Déclaration

* Exemple :

```python
    Dict = {'Tim': 18,'Charlie':12,'Tiffany':22,'Robert':25}
    print((Dict['Tiffany']))

    >>> 22
```

## Trouver une clé

* Vérifier la présence d'une clé :
```python
    if key in dict:
        print(dict[key])
```
